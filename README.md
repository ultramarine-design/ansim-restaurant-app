# 안심식당 앱 (PWA)

농식품부 안심식당 공공데이터를 지역으로 찾아보는 모바일 웹 앱이다. 아이폰 사파리에서 홈 화면에 추가하면 일반 앱처럼 쓴다.

## 무엇을 하는가

1. 첫 화면에서 시도(17개)를 고른다.
2. 시군구 칩과 업종 필터, 이름·주소 검색으로 그 지역 안심식당을 좁힌다.
3. 식당 카드에서 전화 걸기와 네이버 지역검색으로 바로 넘어간다.
4. 한 번 본 지역 데이터는 기기에 캐시돼 오프라인에서도 목록 검색이 된다.

## 데이터

- 출처: 농림축산식품부 안심식당 공공데이터. 기준일 2025-07-09.
- 원본: ../ansim-restaurant-briefing/cache/전국_안심식당.csv (67,581건).
- 앱에는 유효(RELAX_USE_YN=Y) 43,221건만 넣는다. 취소된 식당은 제외.
- 시도별 17개 JSON(data/sido-00.json ~ sido-16.json)으로 쪼개, 그 지역을 고를 때만 받는다. 첫 로딩을 가볍게 하려는 구성이다.
- data/index.json: 시도 목록과 시군구별 건수, 기준일을 담은 색인.
- 휴대폰 번호는 공공데이터에서 010-****-**** 로 마스킹돼 있어 걸 수 없다. 마스킹된 번호는 전화 버튼을 숨긴다. 유선 번호만 전화 버튼이 뜬다.

## 파일 구조

```
안심식당-앱/
  index.html              앱 셸
  app.css                 스타일 (라이트/다크 자동)
  app.js                  로직 (라우팅, 검색, 필터)
  manifest.webmanifest    PWA 설정
  sw.js                   서비스워커 (오프라인 캐시)
  icons/                  앱 아이콘 (192/512/maskable/apple-touch)
  data/                   index.json + 시도별 17개 JSON
  build_data.py           CSV를 data/ JSON으로 변환하는 스크립트
```

## 데이터 갱신 방법

1. data.go.kr 에서 안심식당 CSV를 새로 받아 ../ansim-restaurant-briefing/cache/전국_안심식당.csv 를 교체한다.
2. `python3 build_data.py` 를 돌린다. data/ 의 JSON이 다시 만들어진다.
3. sw.js 의 SHELL/DATA 버전 숫자를 하나 올린다(예: v1 -> v2). 그래야 기존 사용자 기기에서 새 데이터로 갱신된다.
4. 깃에 커밋하고 푸시한다. GitHub Pages가 같은 주소에 자동 반영한다.

## 배포 (GitHub Pages)

정적 파일만 있으므로 빌드 단계가 없다. main 브랜치를 Pages 소스로 두면 끝이다.
설치는 사용자가 아이폰 사파리에서 주소를 열고 공유 - 홈 화면에 추가를 누른다.

## 만든 기준

다른 맛집 검색 앱(지역 드릴다운, 업종 칩 필터, 카드형 결과, 원탭 액션) 패턴을 참고했다.
디자인은 안심을 주제로 한 차분한 편집형. 주조색 소나무 그린, 본문 Pretendard, 제목 Gowun Batang.
위치 기반(내 주변) 정렬과 지도는 좌표 지오코딩이 필요해 2차로 미뤘다.
